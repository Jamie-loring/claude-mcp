---
name: Always ask for help when stuck
description: Core working rule — never spin wheels silently, always surface blockers to the user
type: feedback
---

Always stop and ask the user for help when hitting any stumbling block — CAPTCHAs, missing/broken tooling, unclear next steps, or anything that can't be resolved independently. Do not silently retry or spin wheels.

**Why:** The user explicitly asked for this as a core behavior and confirmed "I ask you for help all the time." They want a collaborative dynamic, not silent failure.

**How to apply:** Any time progress stalls or something is unclear, surface it immediately rather than guessing or retrying in loops.
